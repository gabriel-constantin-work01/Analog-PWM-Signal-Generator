## Analog-PWM-Signal-Generator
Designed a 9kHz Analog PWM Signal Generator with adjustable output amplitude (1.5V - 7V) and duty cycle (40% - 60%).

## Overview
This project presents the design of an analog PWM signal generator using operational amplifiers.

# Key Features

* Adjustable Duty Cycle (40% - 60%)
* Adjustable Output Amplitude (1.5V - 7V)
* Fully Analog Implementation

## How it works

The PWM signal is generated by comparing a reference voltage ($V_{REF}$) with a triangular waveform.

The triangular signal is applied to the non-inverting input of a comparator, while $V_{REF}$ is applied to the inverting input.

- When $V_{triangle} > V_{REF}$ → output is HIGH  
- When $V_{triangle} < V_{REF}$ → output is LOW  

By adjusting $V_{REF}$ using a potentiometer, the duty cycle of the PWM signal can be controlled.

## Calculations

* Threshold Voltages

$$
V_{PS} = V_{CC} \cdot \frac{R_1}{R_2}
$$

$$
V_{PJ} = - V_{CC} \cdot \frac{R_1}{R_2}
$$

* Total period and Capacitor Charging Voltage

$$
f = \frac{1}{T}
$$

$$
T = 4 \cdot \frac{R_1}{R_2} \cdot R_3 \cdot C
$$
  
$$
V_T(t) = V_C(0) + \frac{1}{C} \int i_C dt
$$

* Duty Cycle control

$$
V_{REF} = V_{PS} - (V_{PS} - V_{PJ})(1 - fu)
$$

* Output Voltage Control

$$
V_{out} = \frac{R_{12}}{R_{12} + \alpha P_{2} + R_{14}} \cdot V_{CC}
$$
