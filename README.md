## Analog-PWM-Signal-Generator
Designed a 9kHz Analog PWM Signal Generator with adjustable output amplitude (1.5V - 7V) and duty cycle (40% - 60%).

## Overview
Designe Analog PWM Signal Generator using operational amplifiers.

# Key Features

* Adjustable Duty Cycle (40% - 60%)
* Adjustable Output Amplitude (1.5V - 7V)
* Fully Analog Implementation

## How does it work

The generation of the PWM Signal is achieved by comparing a reference voltage with a triungular signal generated by a comparator-integrator block. The triangular signal is applied to the non-inverting input of an operational amplifier, while the reference voltahe is applied to the inverting input. As long as Vtriangle > Vref, the comparator output will be HIGH, and when Vtriangle < Vref, the output will be LOW. By adjusting Vref using a potentiometer, the pulse width of the signal can be increased or decreased.
